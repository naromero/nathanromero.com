AppDirect UI Framework

Overview

The AppDirect UI Framework is a front-end framework that’s shared amongst designers, front-end developers and back-end engineers at AppDirect. Similar to Twitter Bootstrap, the UI Framework is meant to serve as a library of UI components, patterns and rules to build out features and products at AppDirect.

My Role

UI Framework was more than a year long effort to standardize the look, feel and experience of the AppDirect product. I was one of two designers on the project and spent the better part of my time, designing, documenting and overseeing the implementation of all components that went into the framework. With this in mind, it was necessary to make sure that every component that went in, needed to play well with the framework as a whole.
